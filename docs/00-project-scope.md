# Project Scope

## Objective

Create an open-source American kei-truck-style EV that prioritizes accessible parts, repeatable fabrication, and community contribution.

## Initial contribution focus

The first meaningful contribution should be a well-structured, evidence-backed subsystem package:

- Battery candidate matrix.
- Battery bay envelope.
- HV safety architecture.
- Supplier verification workflow.
- Motor/inverter compatibility notes.
- Whole-vehicle component sourcing matrix.
- CAD packaging assumptions.
- Decision log.

## Design philosophy

1. Use complicated OEM/commercial components where possible.
2. Keep custom fabrication simple and repeatable.
3. Track facts, decisions, and discussion separately.
4. Avoid locking the chassis to one battery too early.
5. Make every part traceable to a source, quote, CAD file, or test.
