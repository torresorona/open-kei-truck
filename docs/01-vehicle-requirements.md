# Vehicle Requirements Draft

| Requirement | Target |
|---|---:|
| Use case | Local utility / light commercial / community prototype |
| Battery chemistry | LFP preferred |
| Battery energy | 35–55 kWh target |
| Traction bus | 300–430 V operating window |
| Battery output | >=60 kW continuous, >=120 kW peak |
| Packaging | Underfloor or under-bed pack preferred |
| Fabrication | Standard tube, sheet, tab-and-slot, bolt-on modules |
| Serviceability | Service disconnect, labeled HV components, accessible PDU/BMS |
| Open-source priority | Repeatable sourcing and documentation over one-off optimization |

## Do not assume yet

- Street legality path.
- Final curb weight.
- Final motor/inverter.
- Final battery supplier.
- Final chassis dimensions.
