# HV Safety Architecture Draft

This is not a complete safety design. It is a checklist of features the open-source project should track before any high-voltage build.

## Required or strongly preferred

- Main positive and negative contactors.
- Precharge circuit.
- Main fuse.
- Manual service disconnect.
- HVIL loop.
- Isolation monitoring.
- BMS with cell voltage and temperature monitoring.
- CAN interface to VCU/inverter/charger.
- Orange HV cabling and proper shielding/strain relief.
- IP-rated connectors.
- Pack venting strategy if applicable.
- Pack crash/impact protection.
- Clearly labeled HV components.
- Lockout/tagout procedure.
- Emergency shutdown procedure.
- Cooling flow and leak monitoring.
- Charger top-voltage limit.
